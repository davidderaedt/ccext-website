CC Extensibility Helpers

Augmenting Adobe's creative tools with the open web

You can now use HTML, CSS and javascript to create extension panels for Adobe Creative Cloud applications such as Photoshop, Illustrator and Premiere.

The following projects are free and open source tools and utilities to help you build those web standard based extensions.


Extension Builder for Brackets

A Brackets extension to easily create Creative Cloud extensions.

Extension Boilerplate

Start your extension from a selection of well formed templates.

CCExt Command line tools

Command line tools to help you set up your system and deploy extensions.

Sample extensions

Sample extensions for various creative tools to help developers get started with real projects.
